FAQ

How long does a typical project take? +
It depends on project scope. A simple landing page takes 1–2 weeks, while a full web application typically takes 4–12 weeks. I'll give you a detailed timeline estimate after our initial discovery call.
Do you work with international clients? +
Yes! I work with clients globally. I'm comfortable communicating in English and Uzbekian, and I adapt to different time zones for meetings and updates.
What is your payment process? +
I typically work with a 50% deposit upfront and 50% upon project completion. For larger projects, we can set up milestone-based payments. I accept bank transfer, PayPal, and Stripe.
Do you provide post-launch support? +
Yes — I offer 30 days of free bug-fix support after launch. Ongoing maintenance and feature development can be arranged via a monthly retainer plan.
Can you work with an existing codebase? +
Absolutely. I regularly take on legacy code optimization, refactoring, and feature additions. I'll do a brief code audit first to assess the current state and plan the work accordingly.
What technologies do you specialize in? +
My core stack is React, Next.js, TypeScript, and Tailwind CSS on the frontend. For backends I work with Node.js and REST/GraphQL APIs. I'm also proficient with Figma for design handoff.
